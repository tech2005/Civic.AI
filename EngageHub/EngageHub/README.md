#  Complaint Routing System

## Overview

Complaint Routing System is a web-based application that allows users to register complaints and automatically routes them to the appropriate department for resolution.
It helps in managing complaints efficiently and tracking their status.

---

##  Features

*  Register complaints easily
*  Automatic routing to departments
*  Track complaint status (Pending / Resolved)
*  Admin panel for managing complaints
*  User authentication

---

##  Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
